# Wafflemaker Design Guidelines

## General Principles
- Cottagecore aesthetic: warm, handmade, cozy
- Fonts: display font for headings, clean sans for body
- Color palette: cream, waffle/golden, syrup/brown, butter
- Dashed borders, rounded cards, gingham background

## Notifications
- Notification bell/icons: use text "Notifications" label, not emoji
- Push notification titles: plain text, no emoji prefixes
- Inline notification toggles expand from a text button, not an icon

## Typography
- Humanist font feel (per Emily's feedback)
- 2.5D buttons with depth/shadow
- Counter-retro style for streak numbers

## Mobile
- PWA-first: standalone display, theme color #C4956A
- Touch targets: minimum 44px
- Safe area padding for bottom nav (safe-b class)
